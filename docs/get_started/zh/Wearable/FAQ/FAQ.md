## T-Embed-CC1101

> 如果需要我们提供帮助请将下载出厂固件的过程录制为视频，并且将对应串口打印的信息也提供给我们。

### 设备无法开机\屏幕黑屏
请先按以下步骤自检:
- 换 Type-C 数据线与电源适配器，确认供电正常；
- 将设备连接至电脑，检查是否可被识别.
- 若可识别，请使用 GitHub 测试程序重新烧录固件;操作步骤可以参考[下载说明](https://github.com/Xinyuan-LilyGO/T-Embed-CC1101/blob/master/docs/flash_download_tool.md), 
- 重启设备确认是否恢复启动.



### NFC无法工作

烧录回我们固件之后，打开NFC界面和我们配送的NFC标签作为检测对象，如果无法识别，请检查以下事项:
- 标签是否正确贴好；
- 标签是否与设备正确连接；
- 设备是否正确烧录了正确的固件；
- 设备是否正确开启了NFC功能；
